Why KislayPHP
Everything you need,
nothing you don't

KislayPHP brings production-grade microservice infrastructure directly into PHP as native C++ extensions—no separate servers, no Docker overhead for the runtime itself.

🚀
C++ Performance, PHP API
Built on embedded HTTP with a multi-threaded worker pool. 63K plaintext req/s and 48K JSON req/s with full PHP request processing—no fast-path bypass.
🔄
Native Async Architecture
Promise-based async with a scalable background worker pool. async(fn) offloads work to C++ threads. Non-blocking HTTP client with automatic retries.
🛡️
Enterprise Gateway
API gateway with rate limiting, circuit breaker, JWT auth, service routing, and dynamic resolver callbacks. Zero-dependency reverse proxy in PHP.
🕵️
Service Discovery
Built-in service registry with health checking and DNS-style resolution. Microservices find each other without Consul or etcd.
📊
Built-in Observability
Counter, gauge, histogram, and timer metrics with Prometheus-compatible export. Zero instrumentation overhead via C++ atomics.
📦
One-Command Install
Installable via PIE (PHP Extension Installer). Enable with a single extension= line in php.ini. Works on Linux, macOS, and Windows.
Full Ecosystem
8 extensions, one cohesive platform

Each extension is a focused, production-ready module. Use them individually or as a full service mesh.

🌐

core

Embedded HTTP server with routing, middleware, HTTPS/TLS, async workers, and Promise API. The foundation for all PHP microservices.

pie install kislayphp/core
🚦

gateway

API gateway with dynamic routing, rate limiting, circuit breaker, JWT auth, and resolver callbacks. Drop-in reverse proxy for PHP.

pie install kislayphp/gateway
🔍

discovery

Service registry and discovery with health checks. Services register themselves; gateway and core resolve them by name automatically.

pie install kislayphp/discovery
📈

metrics

Application metrics with counter, gauge, histogram, and timer types. Prometheus-compatible export endpoint. Zero instrumentation tax.

pie install kislayphp/metrics
📬

queue

In-process message queue with adapter interface. Deterministic sync API for reliable task processing in long-lived PHP workers.

pie install kislayphp/queue
💾

persistence

Data persistence layer with per-request lifecycle hooks. Automatic transaction management when attached to a core App instance.

pie install kislayphp/persistence
⚡

eventbus

Realtime event transport with WebSocket-style rooms and fanout. Server + Socket API with join/leave/emit for live push applications.

pie install kislayphp/eventbus
⚙️

config

Centralized configuration management for distributed PHP services. Environment-aware, hot-reloadable, and namespace-scoped.

pie install kislayphp/config
Code Examples
Build real services fast

From a simple HTTP server to a full microservice stack in minutes.

HTTP Server
API Gateway
Service Mesh
// 5-line PHP HTTP server at 63K req/s
$app = new Kislay\Core\App();
$app->setOption('threads', 4);

$app->get('/api/users', function ($req, $res) {
    $res->json(['users' => User::all()]);
});

$app->post('/api/users', function ($req, $res) {
    $data = $req->getJson();
    $res->json(['id' => User::create($data), 'status' => 'created'], 201);
});

// Non-blocking async offload to C++ worker pool
$app->get('/api/report', function ($req, $res) {
    async(function() { return generate_report(); })
        ->then(function($pdf) use ($res) {
            $res->send($pdf, 'application/pdf');
        });
});

$app->listen('0.0.0.0', 8080); // blocks until stopped

Quick Start
Running in under 5 minutes

From zero to a production HTTP microservice in three steps.

1

Install via PIE

Use the official PHP Extension Installer for one-command install.

pie install kislayphp/core
2

Enable Extension

Add to your php.ini or php -d for immediate use.

extension=kislayphp_extension.so
3

Write Your Service

Full PHP API. Routing, middleware, JSON, async — everything in pure PHP.

php server.php
4

Add the Full Stack

Layer in gateway, discovery, metrics, queue as your service grows.

pie install kislayphp/gateway
